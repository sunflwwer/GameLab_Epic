# 🚀 적 체력바 자동 생성 가이드

## ✨ 개요
이제 **클릭 한 번**으로 모든 적에게 체력바를 추가할 수 있습니다!

---

## 📋 3가지 사용 방법

### 방법 1: Unity 메뉴에서 한 번에 추가 (가장 빠름!) ⭐

1. **Unity 상단 메뉴 → Tools → 적 체력바 → 씬의 모든 적에게 체력바 추가**
2. 완료! 🎉

> 씬에 있는 모든 적(EnemyHealth2D가 있는 오브젝트)에게 자동으로 체력바가 추가됩니다.

---

### 방법 2: 선택한 적들만 추가

1. **Hierarchy에서 적 오브젝트들을 선택** (Ctrl+클릭으로 여러 개 선택 가능)
2. **Unity 상단 메뉴 → Tools → 적 체력바 → 선택한 적들에게 체력바 추가**
3. 완료! 🎉

---

### 방법 3: 적 오브젝트에 직접 스크립트 추가

1. **적 오브젝트 선택**
2. **Inspector → Add Component → EnemyHealthBarAutoSetup**
3. **게임 실행 시 자동으로 체력바 생성**

또는

1. **적 오브젝트 선택**
2. **EnemyHealthBarAutoSetup 컴포넌트 우클릭**
3. **"체력바 생성" 클릭** (에디터에서 즉시 생성)

---

## 🎨 설정 옵션 (Inspector)

### 자동 생성 설정
- **Auto Setup On Start**: ✅ 게임 시작 시 자동 생성
- **Offset**: 적으로부터 떨어진 거리 (Y: 1.5 = 머리 위 1.5 유닛)

### 체력바 크기
- **Bar Width**: 1.0 (체력바 너비)
- **Bar Height**: 0.15 (체력바 높이)

### 색상
- **Background Color**: 검은색 (배경)
- **Full Color**: 초록색 (50% 이상)
- **Half Color**: 노란색 (25~50%)
- **Low Color**: 빨간색 (25% 미만)

### 옵션
- **Hide When Full**: ✅ 풀피일 때 숨김
- **Always Face Camera**: ✅ 카메라 바라보기

## 🛠️ Unity 메뉴 전체 목록

### Tools → 적 체력바 메뉴:

1. **씬의 모든 적에게 체력바 추가**
   - 씬의 모든 적에게 자동으로 추가

2. **씬의 모든 체력바 삭제**
   - 모든 체력바 제거 (초기화)

3. **선택한 적들에게 체력바 추가**
   - 선택한 오브젝트들만 추가

4. **선택한 적들의 체력바 삭제**
   - 선택한 오브젝트들만 삭제

## 💡 프리팹에 적용하는 법

### 방법 A: 프리팹 모드에서 적용
1. **Project 창에서 적 프리팹 더블클릭** (프리팹 모드 진입)
2. **Add Component → EnemyHealthBarAutoSetup**
3. **컴포넌트 우클릭 → "체력바 생성"**
4. **Ctrl+S로 저장**
5. **씬의 모든 인스턴스에 자동 반영!**

### 방법 B: 씬에서 적용 후 프리팹 갱신
1. **씬의 적 하나 선택**
2. **체력바 추가** (위 방법 중 하나 사용)
3. **Inspector 상단 → Overrides → Apply All**
4. **모든 인스턴스에 반영!**

## 🔧 문제 해결

### 체력바가 생성되지 않아요
- EnemyHealth2D 컴포넌트가 있는지 확인
- Console 창에서 에러 메시지 확인

### 체력바가 너무 크거나 작아요
- Bar Width/Height 값 조정
- Canvas Scale은 자동으로 0.01로 설정됨

### 체력바가 안 보여요
- Hide When Full 옵션 확인 (데미지를 받아야 나타남)
- Main Camera 태그 확인

### 체력바를 다시 만들고 싶어요
1. **Tools → 적 체력바 → 씬의 모든 체력바 삭제**
2. **Tools → 적 체력바 → 씬의 모든 적에게 체력바 추가**

## 📝 작동 원리

```
EnemyHealthBarAutoSetup (적에 부착)
    ↓
Start() 시 또는 수동 실행 시
    ↓
자동으로 생성:
  - HealthBarCanvas (World Space)
  - Background (검은 배경)
  - Fill (채워지는 체력바)
  - SimpleEnemyHealthBar (업데이트 스크립트)
    ↓
완료! 자동으로 업데이트됨
```
